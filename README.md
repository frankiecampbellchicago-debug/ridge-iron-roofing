# Ridge & Iron Roofing

Static website for Ridge & Iron Roofing — a fictional Chicagoland roofing contractor.

## Files

- `index.html` — main site
- `styles.css` — all styling
- `script.js` — mobile nav, accordion, form, animations
- `.nojekyll` — disables Jekyll on GitHub Pages

## Local preview

Open `index.html` directly in your browser. No build step required.

## Deploy

Push all files to the root of your GitHub repo with GitHub Pages enabled on `main` branch. Or drag the folder into Netlify.
